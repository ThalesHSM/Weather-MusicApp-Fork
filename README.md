# Weather-MusicApp

Project that allows you to search cities, see the temperature and receive a temperature-based music recommendation.


## Features

+ Search for a City
+ See today weather
+ See music recommendation
+ Save music
+ change celsius to fahrenheit

## Technologies

+ React 


## Libraries

+ React-router-dom
+ Axios
+ Styled-components
+ Prettier
+ uuid


## How to start

+ Clone the repository
+ Run yarn
+ Run Yarn start
